Shortcuts (vscode)

    Editing

  • Html set: ! + enter (해당 파일 html 형식으로의 저장 선행)
  • 행 상하 이동: Alt + 방향키
  • 행 추가(위): Shift + Alt + Enter
  • 행 삭제: Shift + Ctrl + K
  • 행 복사 & 추가: Shift + Alt + 상하 방향키
  • 선택확장&축소: Shift + Alt + 좌우 방향키
  • 스크롤 업다운: Alt + pageup/down
  • 상하 다중 커서: Ctrl + Alt + Shift
  • 틀릭 다중 커서: Alt + Click
  • 현대 커서 주석: Shift + Alt + A
  • 현대 행 주석: Ctrl + /
  • 바꾸기: Ctrl + H
  • 다음찾기/이전찾기: F3 / Shift + F3
  • Drag 후 태그: Alt + W
  • Open the Browser: Alt + B

    File

  • 탭간 이동: Ctrl + Pageup/down
  • 현재파일경로복사: Ctrl + K -> p (영어로 변환 후)
  • 현재파일위치열기: Ctrl + K -> r (영어로 변환 후)
  • 현재파일새창열기: Ctrl + K -> o (영어로 변환 후)
